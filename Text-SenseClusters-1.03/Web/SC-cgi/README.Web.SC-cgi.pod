=head1 NAME

README.Web.SC-cgi - [Web Interface] Description of cgi files used in SenseClusters web interface

=head1 Description

All the files under this directory should be placed under a directory 
called SC-cgi which itself is placed under the cgi directory of the
web-server.

eg: For apache web-server's default settings, the files in this directory 
should be placed under cgi-bin/SC-cgi
